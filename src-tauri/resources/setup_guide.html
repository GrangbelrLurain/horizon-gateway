VPN 연결 성공!

Tailscale 내부망으로 정상 접속하셨습니다. 아래 가이드에 따라 모바일 프록시 설정을 완료하세요.

이 페이지는 Tailscale VPN 내부 경로를 통해 PC에 직접 접속하여 로드되었습니다.

모바일 프록시 설정 단계

1
Root CA 인증서 다운로드 & 설치

모바일 기기에서 HTTPS 패킷 모니터링을 허용하기 위해 Watchtower의 인증서를 다운로드해 신뢰 설정을 켭니다.

인증서 다운로드
  • iOS: 인증서 다운로드 → 기기 설정 → [프로파일 다운로드됨] 터치 및 설치 → 일반 → 정보 → 인증서 신뢰 설정 → 신뢰 활성화
  • Android: 인증서 다운로드 → 설정 → 보안 → 기기에 인증서 설치 → CA 인증서 → 경고 승인 후 설치
2
프록시 자동 구성 (PAC) 설정

가장 간편한 자동 설정입니다. 기기의 Wi-Fi 네트워크 편집창에서 아래 주소를 복사해 프록시 자동 구성 URL로 대입하세요.

  • Wi-Fi 상세설정 → 프록시 → [자동] 선택 → 구성 URL 칸에 복사한 주소 대입 후 저장
3
수동 프록시 설정 (대체안)

자동 구성(PAC)을 지원하지 않는 경우, 아래 값을 직접 Wi-Fi 프록시 수동 편집창에 기입합니다.

서버 주소 (IP)
포트 (Port)
